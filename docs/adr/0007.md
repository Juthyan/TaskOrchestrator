# ADR-0007 — MiddleWare

## Status
Accepted

## Context
To deal with the exception and to avoid to the same code
it the api. We created a middleware to cath them.
Without the middleware, DomainException was exposing a full stack trace to the client.
Each endpoint would need its own try/catch block, leading to code repetition.

## Decision
We implement a middleware

## Reasons
- More maintanable and more readanle
- we dont repeat ourself

## Consequences
- the middleware catch the exception and deal with them
depending of the kind of exception
